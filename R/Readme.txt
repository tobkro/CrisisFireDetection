These scripts are used to obtain information from Landsat, calculate the NBR or NDVI from a certain extent, and test for pixel timeline anomalies on certain points supplied by a shapefile.

All these files are contained in this directory for use in the case of January 2016s conflict in central darfur.

For more information about this conflict, see 

http://darfurconflict2016.amnesty.org/report/5
